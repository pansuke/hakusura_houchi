| ID | 決定 |

| M3-005 | 最古カードだけを使用判定し、使用不能なら後続カードを探索しない |
| M3-006 | `grant_card_play` Effectで追加カード使用権を定義する |
| M3-007 | 調整値をBattleRuleConfig化し、開発ViewerからAPI経由で編集・保存する |

| M3-009 | 暫定で自分の行動機会を3回失った後に復活する |
| M3-010 | 復活時はHP / MP最大、Deck再shuffle、初期手札3、discard空、Draw Gauge 0 |
| M3-011 | 復活位置は自陣Nexus。移動で対面したActionはそのままカード行動可能 |
| M3-012 | 各Characterが個別位置を持つ。範囲は-1000～1000 |
| M3-013 | PUSHはCharacter stat。50前後を高値の暫定目安とする |
| M3-014 | 非対面時はPUSH全量、対面時はPUSH差で進軍する |
| M3-015 | Damageはbase + standardized stat scaling |
| M3-016 | physicalはAR、magicはMR、trueは軽減なし |
| M3-017 | 防御式はLoL型`100 / (100 + defense)` |
| M3-018 | 中間丸めなし、最終切り捨て、最低Damage 1 |
| M3-019 | Local / Adjacent / GlobalをM3-Aで同時実装する |
| M3-020 | TOP / BOTのAdjacent先はMID。MIDはTOP / BOTの有効対象からSeed付きrandom |
| M3-021 | Adjacentに有効な敵Characterがいなければ不発。Nexusへfallbackしない |
| M3-022 | GlobalはTOP / MID / BOTの生存敵Character全員だけを対象とする |
| M3-023 | GlobalはNexusを対象にしない |
| M3-024 | Localは敵Nexus位置へ到達し、対面相手がいない場合だけNexusを対象にする |
| M3-025 | Nexus攻撃は専用攻撃ではなく通常Damageカードを使用する |
| M3-026 | Nexus最大HPは暫定8000 |

| M3-028 | 本番BattleはNexus破壊まで継続し、最大Action勝敗を設けない |
| M3-029 | Simulation Safety Limit到達はdrawではなくerror |
| M3-030 | 進軍専用カード分類は作らない |

# 3. 文書間の主な修正

| 旧記載 | 更新後 |
|---|---|
| 距離・座標を扱わない | 2D詳細座標は扱わず、レーン内の抽象1次元位置だけを扱う |
| 進軍は単一の支配度 | 各Characterの個別位置とPUSHで表現する |
| M3-BでLocal / Adjacent / Global | 対面・位置と同時に必要なためM3-Aへ移動 |
| M3-Bで本番最大Action | 本番最大Actionは廃止。Safety Limitはerror |
| shuffleなし / hand 5 | M1暫定。M3ではshuffle、initial 3、max 7 |
| 使用可能カードを先頭から探索 | M3では最古カードだけを判定する |
| 全滅勝敗 | M3-BではNexus破壊だけが勝敗条件 |
| 進軍専用Effect候補 | 専用分類は作らない |
| Global対象未定 | 生存敵Character全員、Nexus除外 |
| Adjacent対象なし未定 | 不発、Nexus除外 |
