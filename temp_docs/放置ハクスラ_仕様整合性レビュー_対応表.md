# 4. Product Ownerへ残したDecision

以下はゲーム体験へ大きく影響するため、自動決定していない。

- 本番3レーンAction Scheduler
- 本番Deckのshuffle・循環・手札値
- 復活テンポと復帰状態
- 反撃・追撃・割り込みのAction扱い

いずれもDecision Gate、決定期限、影響マイルストーンをWBSへ明記した。

現在のM0-B、M1、M2の実装着手には影響しない。
